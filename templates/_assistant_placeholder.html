{# Empty assistant bubble. Opens an SSE connection on insert and handles three named events (sse-swap="token,done,error"): - `token`: each chunk's HTML-escaped text gets appended inside this div (`hx-swap="beforeend"`), so the response materializes as it streams. - `done`: the server sends the final persisted message bubble with `hx-swap-oob` set. HTMX takes that out-of-band and uses it to replace this whole placeholder — the streaming class goes away and the row gets a stable `id="message-{N}"`. - `error`: a small error fragment, appended like a token so it surfaces where the user was expecting the assistant reply. #}
