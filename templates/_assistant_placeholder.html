Opens an SSE connection on insert and handles these named events: - `token`: each chunk's HTML-escaped text gets appended inside this div (`hx-swap="beforeend"`), so the response materializes as it streams. - `done`: the server sends the final persisted message bubble with `hx-swap-oob` set. HTMX takes that out-of-band and uses it to replace this whole placeholder — the streaming class goes away and the row gets a stable `id="message-{N}"`. The same payload carries OOB updates for the per-turn tool-usage card (summary swap to past tense + freeze of any unpaired rows). - `error`: a small error fragment, appended like a token so it surfaces where the user was expecting the assistant reply. - `title` (phase 11d): the renamed sidebar `
  • ` with `hx-swap-oob="true"` on its root. HTMX takes it OOB and replaces the matching `#chat-{id}` row in the sidebar — the placeholder itself is untouched. Fires BEFORE done so the connection is still alive when it arrives. - `tool-call` / `tool-result` (phase 12e): OOB fragments that build up the aggregated tool-usage card above this placeholder. The first tool-call inserts the card via `beforebegin:#assistant-stream-{id}`; subsequent tool-calls append rows + bump the summary count; each tool-result OOB-replaces its row with a frozen variant. Events not emitted for a given turn are silently ignored, so listing them all here is safe whether or not the turn used tools. #}
