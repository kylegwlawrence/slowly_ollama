{# One-time banner explaining that the auto-titler is disabled because the title model isn't installed. Delivered as an SSE `title-warning` event from the streaming endpoint when `ollama.generate_title` raises `OllamaModelMissing`. The root `hx-swap-oob="true"` is critical — without it HTMX would dump this into the streaming assistant placeholder. With it, HTMX finds the element with matching id (`#title-warning` in the sidebar footer of index.html) and replaces it in place. Per-session de-dup with NO visual flash: the fragment is rendered `hidden` (HTML5 attribute → display:none) and the `htmx:load` handler decides whether to reveal it. - FIRST landing in a session: sessionStorage flag is unset; we set it AND clear `hidden`, so the banner becomes visible. - LATER landings: the flag is already set; we call this.remove() while the element is still hidden, so the user sees nothing. Closing the tab clears sessionStorage, so the user gets exactly one banner per session if the model is still missing. #}
info Auto-titles disabled — install tinyllama:1.1b-chat-v1-fp16 in Ollama to enable.
